## 2025-05-15 - [ARIA and State Sync in Mobile Navigation]
**Learning:** In projects using vanilla JS/CSS, consistency between IDs and class names used in JS and those defined in HTML/CSS is critical. Accessibility improvements like `aria-expanded` and `aria-controls` must be synchronized with the visual state changes.
**Action:** Always verify that JS selectors and class toggles match the existing HTML structure and CSS definitions before renaming them for "consistency" alone, as it can break functionality if not applied globally.
